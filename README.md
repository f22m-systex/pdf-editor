# PDF 套印編輯器 (Web 核心版) 📄✨

這是一套專為「PDF 表單套印」設計的視覺化定義檔編輯工具。

本專案成功將傳統的桌面端套印編輯器升級為現代化的 Web 應用程式。使用者可以直接在瀏覽器中載入多頁 PDF 表單，透過直覺的拖曳、對齊與快捷鍵操作，快速定義各欄位的位置與格式，並最終匯出交由後端 (如 Java 模組) 執行實際套印的 XML 定義檔。

## 🌟 核心特色

* **無依賴、即開即用 (Portable)**
  採用單一 HTML 檔案架構設計，無需複雜的 Node.js 環境或編譯過程。只要雙擊檔案或部署至 GitHub Pages，即可在任何現代瀏覽器中流暢運行。
* **原生多頁 PDF 支援**
  內建 Mozilla PDF.js 引擎，支援真實多頁 PDF 載入與跨頁結構導覽，告別只能單張轉圖編輯的痛點。
* **專業級視覺編輯體驗**
  * **精準定位**：支援 10pt 網格吸附，以及 `Ctrl` / `Shift` 配合方向鍵的 1pt / 5pt / 10pt 無段微調。
  * **批次處理**：支援滑鼠框選、多重選取，並內建「齊左/齊右/上下置中」等多種對齊工具 (支援快捷鍵操作)。
  * **無損縮放**：採用純 CSS 轉換技術 (`transform: scale`) 進行畫布縮放，確保屬性面板的座標永遠精準對應 A4 實際 pt 單位。
* **沉浸式資料預覽模式 (Preview Mode)**
  打破「只見座標不見資料」的盲區。一鍵切換預覽模式，並支援整批 JSON 測試資料匯入 (Mock Data)。提供「列印實況」、「螢光筆」、「校對紅框」三種檢視風格，確保最終輸出的排版完美無瑕。

## 🛠️ 技術棧 (Tech Stack)

* **UI & 佈局**：HTML5, Vanilla JS, [Tailwind CSS](https://tailwindcss.com/) (CDN)
* **畫布與渲染引擎**：[Fabric.js](http://fabricjs.com/) (負責元件拖曳、縮放與選取邏輯)
* **文件解析引擎**：[PDF.js](https://mozilla.github.io/pdf.js/) (負責背景多頁 PDF 高解析度渲染)

## 🚀 快速上手 (Quick Start)

1. 下載本專案的 `index.html` 檔案。
2. 使用任何現代瀏覽器 (Chrome, Edge, Safari) 雙擊開啟。
3. 點擊頂部工具列的 **「載入底圖」** 上傳您的 PDF 表單。
4. 點擊 **「+ 新增欄位」** 或使用右鍵開始您的套印定位作業！
